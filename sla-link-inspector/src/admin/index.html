Configuration

Configure when and how to notify about linked-issue SLA status.

Requirements

The panel shows SLA only for linked issues that the viewing user can access. Jira enforces project permissions: if a user has no access to a linked issue’s project, that issue will appear as “Error loading” in the panel. To see SLAs for links into other queues or projects, users need at least browse access to those projects.

Custom message template

Customize the comment text. Use variables:

  • {{issueKey}} – linked issue key
  • {{slaName}} – SLA field name
  • {{remainingTime}} – e.g. “9d left” or “2h overdue”
  • {{status}} – at risk / breached / 30 minutes remaining
  • {{assignee}} – @mention(s): assignee, reporter, watchers as configured for that trigger
